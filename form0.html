O atributo type="text" específica o valor que o input irá receber.

Type="button" cria um input de botão para um evento. Precisa utilizar JS

Type="Submit" utilizado paras envios de dados, sem necessidade de JS


O type="checkbox" é usado para dar mais de uma opção para o usuário.

O type="date" é usada para gerar uma caixa que permite o usuário inserir uma data.



O type="email" identificca se o campo foi preenchido com @ e . porém não faz validação.



O type="file" envio de arquivo.



O type="number" determina o valor recebido pelo input como numérico.



O type="password" protege os dados digitados no formulário.

o type="radio" serve para dar apenas uma opção para o usuário.

Type="range" gera uma opção dinâmica horizontal, EX: partindo do ponto A para o ponto B.

Gera um campo de busca dentro do seu site.

O type="tel" usado para a entrada de valor númerico
